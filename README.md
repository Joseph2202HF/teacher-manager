# GestionEns

Application web de gestion des enseignants développée avec React, PHP et MySQL.

## Fonctionnalités

* Authentification
* Gestion des enseignants (CRUD)
* Recherche et filtrage
* Calcul automatique des salaires
* Tableau de bord statistique

## Technologies

**Frontend :** React, Vite, Tailwind CSS, Axios

**Backend :** PHP, API REST, MySQL, JWT
